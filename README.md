# goit-markup-hw-02

Домашнє завдання №2 курсу GoIT "HTML/CSS" — верстка лендингу WebStudio.

## Структура проєкту

```
goit-markup-hw-02/
├── css/
│   ├── common.css
│   ├── header.css
│   ├── hero.css
│   ├── features.css
│   ├── team.css
│   ├── portfolio.css
│   ├── footer.css
│   └── main.css
├── images/
│   ├── team-1.jpg ... team-4.jpg
│   └── portfolio-1.jpg ... portfolio-6.jpg
├── index.html
├── .gitignore
├── .prettierrc
└── README.md
```

## Розділи сторінки

- Header (логотип, навігація, контакти)
- Hero (заголовок, кнопка Order Service)
- Our Features
- Our Team
- Our Portfolio
- Footer

## Технології

- Семантична HTML5-розмітка
- CSS (Flexbox), без `!important`, тільки селектори класів
- Google Fonts: Raleway + Roboto (одним посиланням)
- modern-normalize (мінімізована версія)
- Prettier
